
# VerifyAI – Fake News Detection System

## Overview
VerifyAI is an AI-powered system that detects fake news, verifies sources, and explains results in real-time.

## Features
- Real-time fake news detection  
- Explainable AI  
- Multi-source verification  
- Browser extension support  

## Tech Stack
- React.js  
- FastAPI  
- MongoDB  
- HuggingFace Transformers
